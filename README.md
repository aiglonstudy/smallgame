# 坦克大战 - Tank Battle

经典坦克大战游戏的 HTML5 实现。纯前端，打开即玩。

## 操作方式

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → / WASD | 移动坦克 |
| Space | 射击 |

## 游戏规则

- 保护基地（屏幕下方的鹰旗），防止被敌人摧毁
- 消灭所有敌人即可过关
- 每关有 20 个敌人，从屏幕上方分波出现
- 玩家有 3 条命
- 砖墙可被子弹摧毁，钢墙不可摧毁
- 关卡越高，敌人越强（速度更快、生命值更高）

## 运行方式

直接用浏览器打开 `index.html` 即可游玩。

## 关卡说明

- 地图由砖墙和钢墙构成，基地周围有保护墙
- 敌人从上方 3 个刷新点出现
- 玩家从底部中央出发
